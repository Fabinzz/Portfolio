(PI4) Mistergold

MisterGold é um e-commerce de joias, oferece produtos como brincos, anéis e colares.

Fui responsável pelo desenvolvimento do frontend e parte da integração com backend, utilizei HTML, CSS e Material Design 3 para agilizar o desenvolvimento das telas. Auxiliei na integração com o backend, utilizando Javascript para interagir com os endpoints REST. O backend foi implementado em Java com Spring Boot, banco de dados MongoDB, recurso de upload de imagens, envio de e-mails para clientes e hospedagem na AWS.

(PI5) BemTiVi Petshop

Bemtivi Petshop é um sistema de gestão de petshop, composto por dois aplicativos: um voltado para o administrador, para cadastrar e gerenciar produtos, serviços e clientes; e outro voltado para o cliente, onde ele pode visualizar produtos, agendar serviços e realizar compras.

Neste projeto, desenvolvi telas e alguns componentes para o frontend além de auxiliar na integração com o backend, utilizando TypeScript e React Native. O backend foi implementado em Java com Spring Boot, banco de dados Postgres, gerenciamento de pedidos, e hospedagem do backend na AWS.
